About

What is jackshit?

jackshit is a browser-based 3D viewer for surveying and engineering data. It renders LandXML surfaces, GeoTIFF elevation models, and ASCII Grid DEMs directly in your browser — no plugins, no installs, no server.

Supported Formats

  • LandXML — TIN surfaces with full metadata (CRS, project info, application)
  • GeoTIFF — Raster elevation models (.tif / .tiff)
  • ASCII Grid — ESRI-format grid files (.asc)

How It Works

Files are parsed entirely in your browser. LandXML is processed on the main thread; DEM formats use a Web Worker for non-blocking parsing. Geometry is rendered with Three.js using per-surface centroid offsets to maintain float-precision at any coordinate magnitude.

Tech

  • Three.js r138 (ES module, no bundler)
  • GeoTIFF.js for raster parsing
  • Custom first-person camera controls
  • Static site — hosts on any file server or S3 bucket
